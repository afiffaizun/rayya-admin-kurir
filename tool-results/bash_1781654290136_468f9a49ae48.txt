# UI Design Detailed Description

## 1. Overall Layout and Structure
The UI displays a clean, centered login interface for an admin panel. The layout follows a vertical stack structure with:
- A header section containing the logo, title, and subtitle
- A login form card positioned below the header
- All elements are centered horizontally on a white background
- The design uses ample white space for a minimalist, professional appearance

## 2. Colors Used
- Background: `#FFFFFF` (pure white)
- Primary blue: `#0066FF` (for logo, button, and text)
- Light gray: `#F5F5F5` (for input fields background)
- Dark gray: `#333333` (for labels and text)
- Light blue accent: `#E6F0FF` (subtle background gradient in corners)
- Input border: `#E0E0E0` (light gray)
- Button text: `#FFFFFF` (white)

## 3. Text Content
- "Admin" (large title)
- "Pengantaran Cepat & Higienis" (subtitle)
- "Username" (label)
- "username" (placeholder in input field)
- "PIN / Kata Sandi" (label)
- "Masukkan 6 digit PIN" (placeholder in input field)
- "Lupa PIN?" (link text)
- "Masuk Sekarang" (button text with arrow)

## 4. UI Components
- **Logo**: Blue square with rounded corners containing a white water drop icon
- **Title**: "Admin" text
- **Subtitle**: "Pengantaran Cepat & Higienis" text
- **Login Card**: White container with subtle shadow and rounded corners
- **Username Input**: Text input field with user icon
- **PIN Input**: Text input field with lock icon and eye icon for visibility toggle
- **Forgot PIN Link**: "Lupa PIN?" text link
- **Login Button**: Blue button with white text and right arrow icon

## 5. Spacing and Sizing
- Overall page: Full viewport height with centered content
- Logo: 60px × 60px square
- Title: Large font size, ample margin below
- Subtitle: Smaller font, centered below title
- Login card: 400px width, auto height with padding
- Input fields: Full width of card, 48px height
- Button: Full width of card, 48px height
- Spacing between elements: Generous vertical margins (approximately 24px between sections)

## 6. Typography
- **Title**: Sans-serif font, bold weight, approximately 36px
- **Subtitle**: Sans-serif font, regular weight, approximately 16px
- **Labels**: Sans-serif font, regular weight, approximately 14px
- **Input placeholders**: Sans-serif font, regular weight, approximately 14px, light gray color
- **Button text**: Sans-serif font, medium weight, approximately 16px
- **Link text**: Sans-serif font, regular weight, approximately 14px, blue color

## 7. Visual Effects
- **Shadow**: Subtle shadow on login card (rgba(0,0,0,0.1) with 0-2px offset)
- **Borders**: Light gray border on input fields (1px)
- **Rounded corners**: All cards and inputs have 8px border radius
- **Gradient**: Subtle blue gradient in top-right and bottom-left corners of the page
- **Hover effects**: (implied) Button likely has hover state with slightly darker blue
- **Focus states**: Input fields likely have blue border when focused

## 8. Icon Descriptions
- **Logo icon**: White water drop symbol inside blue square
- **Username icon**: Gray user silhouette icon (person)
- **PIN icon**: Gray lock icon
- **Eye icon**: Gray eye symbol for password visibility toggle

The design emphasizes clarity and simplicity, with a professional blue and white color scheme that suggests reliability and cleanliness, appropriate for an admin interface related to delivery services.
